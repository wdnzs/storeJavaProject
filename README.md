# Projeto Store
Projeto full stack completo! Java Spring Boot no back end e Ionic no front end. Desenvolvido a partir do curso Spring Boot, Hibernate, REST, Ionic, JWT, S3, MySQL, MongoDB do professor Nelio Alves.
